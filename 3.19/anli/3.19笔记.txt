安装JDK：jdk就是java的编译环境
安装jdk的目的就是让我们可以在windows系统
上编写java代码并且运行
安装完 jdk还有其他事要做
1.右击我的电脑，属性，高级系统设置
2.点开path(控制台中能够通过指令执行的程序路劲)
3.C:\Program Files\Java\jdk1.8.0_65\bin 添加到环境变量中
4.java 运行指令	javac 编译指令
编写第一个java程序
新建文本文档  Test.java首字母必须大写
cmd常用指令
cls    清屏
找到我们.java文件所在的文件路径
d:切换根目录
cd 文件夹名 进入新的文件路径
dir:显示当前路径下所有的文件或者文件夹
javac Test.java  编译代码，形成一个可执行文件 如果语法通过的话
会产生一个Test.class文件（可执行文件）
代码发生任何改变一定要保存，然后重新编译
java Test 执行可执行文件
cd cd D:\工作\ZD1901\3.19通过绝对路径进入文件夹
绝对路径：从根目录开始到最后一层文件夹的完整路径
cd ../返回上一级路径
